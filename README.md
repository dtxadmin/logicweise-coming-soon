# Logic Weise Website

Marketing site for Logic Weise LLC focused on commercial project review conversions in Dallas-Fort Worth and North Texas.

## Current pages

- `index.html`: Main commercial homepage focused on access control, CCTV, cabling, fire alarm coordination, and takeover support
- `about.html`: Company profile and commercial positioning page
- `privacy.html`: Privacy policy for project review submissions and site data handling
- `accessibility.html`: Accessibility statement and support information

## Notes

- The site uses static HTML and CSS
- Primary conversion action is `Project Review`
- Contact and project review flows route back to the homepage review section
