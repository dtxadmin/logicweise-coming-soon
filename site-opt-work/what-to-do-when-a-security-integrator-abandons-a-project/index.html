What to Do When a Security Integrator Abandons a Project

Start with risk stabilization, access control of documentation, and a takeover assessment.

Direct Answer

Stabilize immediate operational risk first, preserve all project records, and obtain an independent technical assessment before restarting installation.

Practical Steps

  • Secure credentials, keys, and admin access immediately
  • Collect drawings, change orders, and installed hardware inventory
  • Document open defects with photos and location notes
  • Prioritize life-safety and operations-critical items
  • Build a sequenced takeover plan with milestones
